## Split Shell - Basic terminal multiplexer for zsh

This program basically splits zsh into two parts: the input and output.

scrolling through your past entered commands shows the associated output with it.

## Why? 
Because i think it looks cool.

## Usage
./spsh

## Build
compile command: gcc -O2 -Wall -o spsh spsh.c -lutil -lvterm

## Compatibility
theoretically works with any zle plugin. tested with syntax highlighting

no idea if this will work with bash or any other shell
